# Neural Network From Scratch

A neural network framework built from scratch using NumPy to understand the core concepts behind deep learning, including forward propagation, backpropagation, activation functions, loss functions, and optimization algorithms.

## Features

* Dense Layers
* ReLU, Sigmoid, Tanh, and Softmax Activations
* Categorical Cross Entropy Loss
* SGD and Adam Optimizers
* Sequential Model Architecture
* Accuracy Metrics
* Model Checkpointing
* Xavier and He Initialization
* Dropout Regularization
